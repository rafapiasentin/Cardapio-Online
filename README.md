🍔 ART Burger - Cardápio Digital

Projeto em andamento 🚧
Um cardápio digital desenvolvido com HTML e CSS, inspirado em uma hamburgueria real, com foco em praticar desenvolvimento front-end e criar uma solução que futuramente possa ser utilizada em um negócio real.

📌 Sobre o projeto

O ART Burger é um projeto de site de cardápio digital criado durante as aulas de Front-End, com o objetivo de aplicar na prática conceitos de:

Estruturação com HTML5
Estilização com CSS3
Organização visual de produtos
Construção de layout para negócios do ramo alimentício

A ideia do projeto surgiu como uma forma de desenvolver um site funcional e visualmente agradável, inspirado em uma hamburgueria real, podendo futuramente ser expandido para uso comercial.

🎨 Conceito do projeto

O nome ART Burger traz a proposta de unir arte + gastronomia, refletindo isso tanto no visual quanto nos nomes dos hambúrgueres.

🖼️ Alguns produtos foram inspirados em referências artísticas, criando uma identidade mais criativa para o cardápio.
